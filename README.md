### 1. Задание №1

Входные и выходные данные:
- На вход программа получает 2 целых числа, где первое из них (a0) представляет собой нулевой член последовательности, а второе (n) - количество членов последовательности, которые необходимо вывести (0 < n < 100). 
- Результатом работы программы является вывод на экран последовательности целых чисел.

Алгоритм:
- Закономерность представляет собой зависимость члена последовательности от его номера и описать это можно так. Пусть a - последовательность, i>0 - номер элемента. Тогда:
  - Если i - нечётное, то $a_i$ = -4 * $a_(i-1)$
  - Если же i - чётное, то $a_i$ = 2 + $a_(i-1)$
 
Программа:
```java
import java.util.Scanner;
public class Main {

    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int a0 = in.nextInt();
        int n = in.nextInt();
        for (int i = 0; i < n; i++) {
            if(i%2==0){
                a0*=-4;
                System.out.print(a0 + " ");
            } else {
                a0+=2;
                System.out.print(a0 + " ");
            }
        }
    }
}
```
