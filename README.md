### 1. Задание №1
Условие задачи: сформировать последовательность по её нулевому элементу и заданному правилу.

Входные и выходные данные:
- На вход программа получает 2 целых числа, где первое из них ($a_0$) представляет собой нулевой член последовательности, а второе ($n$) - количество членов последовательности, которые необходимо вывести (0 < n < 100). 
- Результатом работы программы является вывод на экран последовательности целых чисел.

Алгоритм:
- Закономерность представляет собой зависимость члена последовательности от его номера и описать это можно так. Пусть $a$ - последовательность, $i$ >0 - номер элемента. Тогда:
  - Если $i$ - нечётное, то $a_i$ = -4 * $a_(i-1)$
  - Если же $i$ - чётное, то $a_i$ = 2 + $a_(i-1)$
 
Программа:
```java
import java.util.Scanner;
public class Main {

    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int a0 = in.nextInt();
        int n = in.nextInt();
        for (int i = 0; i < n; i++) {
            if(i%2==0){
                a0*=-4;
                System.out.print(a0 + " ");
            } else {
                a0+=2;
                System.out.print(a0 + " ");
            }
        }
    }
}
```

### 2. Задание №2
Условие задачи: Дана последовательность из n целых чисел. Определите, можно ли переставить ее элементы так, чтобы сумма любых двух соседних элементов была нечетной. Выведите "YES" или "NO".

Входные и выходные данные:
- На вход программа получает целочисленное число n - количество элементов, а после - сами n элементов, которые тоже представляют из себя целые числа.
- Результатом работы программы является вывод на экран строки "YES" либо "NO".

Алгоритм:
- Заметим, что сумма двух чисел нечётна только тогда, когда слагаемые различной чётности. Тогда для вывода "YES" необходимо, чтобы количество четных и нечетных чисел отличалось на величину не большую, чем 1 (связано с четным и нечетным n). Иначе программа выведет "NO".
 
Программа:
```java
import java.util.Scanner;
public class Main {
  public static void main(String[] args) {
              Scanner in = new Scanner(System.in);
              int n = in.nextInt(), x, k1=0, k2=0;
              for (int i = 0; i < n; i++) {
                  x = in.nextInt();
                  if(Math.abs(x)%2==0)k1++;
                  else k2++;
              }
              if(Math.abs(k1-k2)<=1) System.out.println("YES");
              else System.out.println("NO");
          }
}
```

### 3. Задание №3
Условие задачи: Полупростое число – это натуральное число, представимое в виде произведения двух простых чисел (не обязательно разных): n = p · q. Примеры полупростых: 4=2·2, 6=2·3, 9=3·3, 10=2·5. Не являются полупростыми: 8=2·2·2, 12=2·2·3. Для каждого входящего числа n требуется вывести наибольшее полупростое не превышающее n (если такого числа не существует, выводите -1).

Входные и выходные данные:
- На вход программа получает последовательность целочисленных чисел, последний элемент которой равен -1.
- Результатом работы программы является вывод на экран последовательности соответствующих приближенных полупростых чисел, если такие существуют(иначе на месте будет -1).

Алгоритм:
- Алгоритм рассматривает все парные делители числа и проверяет, является ли каждый из них простым. Если такая пара подходящих делителей не находится, алгоритм продолжает рассматривать делители числа, заданного на единицу меньше, и продолжает это делать до тех пор, пока это число больше нуля. Если пара в итоге так и не находится, алгоритм выводит -1 на экран и продолжает обрабатывать следующие числа, введенные пользователем, пока не встретится -1.

Программа:
```java
import java.util.Scanner;
public class Main {
    public static boolean pr(int n){
        if(n<2)return false;
        int j = 2;
        boolean f = true;
        while(j*j<=n && f){
            if(n%j==0)f=false;
            j++;
        }
        return f;
    }
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        while(n!=-1){
            boolean f = false;
            while(n>0){
                int j=2;
                while(j*j<=n){
                    if(n%j==0){
                        if(pr(j) && pr(n/j)){
                            System.out.print(n + " ");
                            f = true;
                            n = 0;
                        }
                    }
                    j++;
                }
                n--;
            }
            if(!f) System.out.print(-1 + " ");
            n = in.nextInt();
            }
    }
}
```

### 4. Задание №4
Условие задачи: Дан массив из n чисел. Найти минимальное расстояние между двумя одинаковыми элементами массива или вывести "NO", если его определить невозможно.

Входные и выходные данные:
- На вход программа получает положительное целое число n, после которого вводится массив из n чисел, которые мы будем считать вещественными.
- Результатом работы программы является вывод на экран расстояния, являющегося натуральным числом.

Алгоритм:
- Алгоритм попарно сравнивает каждые 2 элемента массива на равенство (кроме пар сравнения элемента с самим собой). Если элементы равны, то значение сравнивается с изначально заданным минимумом(равным n), и если такое расстояние меньше, то оно записывается в значение минимума. Если после всех сравнений значением минимума так и осталось n, то программа выводит "NO", иначе выводит минимальное искомое расстояние.

Программа:
```java
import java.util.Scanner;
public class Main {
      public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        double []a = new double[n];
        for (int i = 0; i < n; i++) {
            a[i] = in.nextDouble();
        }
        int min=n;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if(Math.abs(a[i]-a[j]) < 1e-9 && i!=j){
                    if(Math.abs(i-j)<min)min=Math.abs(i-j);
                }
            }
        }
        if(min==n) System.out.println("NO");
        else System.out.println(min);
    }
}
```
